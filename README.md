# Avatars
Host  League Avatars
Since you have quite a few files to move, editing them one by one in the browser is going to be a headache. The fastest way to do this without using complex command-line tools is to use the GitHub "Web Editor" trick.

It’s essentially a secret version of VS Code that runs right in your browser.

The Fast "Web Editor" Method
Go to the main page of your Avatars repository.

Press the period key (.) on your keyboard.

Alternatively, change the URL from github.com to github.dev.

A full code editor will open in your browser.

Create your folders first:

Right-click the folder icon in the left sidebar and select New Folder.

Create Midnight Grind, Nordic Warriors, Orion's Belt, and Miscellaneous.

Move the files:

Simply click and drag your images from the main list into the folders you just created. You can hold Ctrl (or Cmd) to select multiple files at once.

Save your work (Crucial):

Look at the left sidebar for the Source Control icon (it looks like a little circle with lines).

Type a message like "Organized avatars into folders."

Click the Commit & Push button at the top.

Why this is the best way:
Bulk Moving: You can move all the "Midnight Riders" files in one big group.

Organization: You can see exactly which files are left in the "main vault" so you don't miss any.

Instant Update: As soon as you hit "Commit," your new index.html script will pick up those folders and start displaying the sections.

One small thing to check:
When you rename or move files into a folder, GitHub creates a "path." If a file was AMG.png and you move it to a folder called Midnight Grind, its new path is Midnight Grind/AMG.png. My script handles this automatically, so you don't have to worry about the code—just get the folders looking the way you want!

Give that period key (.) trick a try; it’s a total game-changer for organizing a "whole bunch" of files.
